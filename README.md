# Dhanashree Industries – Solar Market Analysis (Power BI Dashboard)

## Project Overview

This project presents a **Power BI dashboard analysis of solar rooftop market data** collected during a **Market Research Internship at Dhanshree Industries, Nagpur**.

The objective of this project is to analyze customer demand, rooftop feasibility, and potential solar installations using data visualization tools.

---

## Objectives

* To Analyze solar rooftop market opportunities.
* To Identify potential customers for solar installation.
* To Understand customer requirements and location patterns.
* To Visualize market research data using Power BI.

---

## Tools & Technologies

* Microsoft Power BI
* Microsoft Excel
* Data Cleaning & Visualization
* Market Research Techniques

---

## Dashboard Insights

The dashboard provides insights such as:

* Top locations with solar installation demand
* Customer categories (Residential / Commercial / Industrial)
* Rooftop feasibility analysis
* Lead generation trends
* Potential solar capacity distribution

---

## Dataset

The dataset includes:

* Customer Name
* Location
* Rooftop Type
* Electricity Consumption
* Interest in Solar Installation
* Estimated Solar Capacity

*Note: Data used in this project is sample/simulated data created for learning and analysis purposes by the Company.*

---

## How to Use

1. Download the `.pbix` file from the **Dashboard folder**
2. Open it using **Microsoft Power BI Desktop**
3. Explore the interactive dashboards and visualizations
